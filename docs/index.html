Documentation Application Web Framelab

1. Barre de navigation

La barre de navigation située en haut de l’application permet d’accéder aux principales fonctionnalités.

  • Accueil : permet de revenir à la page d’accueil.
  • Challenge : affiche le challenge actuellement en cours.
  • Participations de la semaine : affiche les participations du challenge en cours.
  • Toutes les participations : permet de consulter toutes les participations.
2. Page d’accueil
  • L’image du challenge en cours
  • Un bouton pour accéder à la page du challenge

Cette page permet de découvrir rapidement le challenge actif.

3. Page du challenge en cours
  • Le thème
  • La description
  • La date de début
  • La date de fin

Actions disponibles :
- Participer
- Voir les participations

4. Participation à un challenge

Chaque utilisateur peut soumettre une seule participation.

  1. Cliquer sur Participer
  2. Sélectionner une photo
  3. Cliquer sur Envoyer
5. Détail d’une participation
  • Nom de l’utilisateur
  • Date de publication
  • Numéro du challenge

Notation :

⭐ Créativité
⭐ Respect du thème
⭐ Technique

Autres actions :
- Ajouter un commentaire
- Voir les notes
- Lire les commentaires
- Suppression (admin uniquement)

6. Toutes les participations

Consultation de toutes les participations.

  • Filtrer par utilisateur
  • Filtrer par challenge
7. Mon compte

Consultation des informations personnelles uniquement.

8. Fonctionnalités Administrateur

Les administrateurs peuvent créer des challenges et supprimer des commentaires.

Création d’un challenge :

  1. Cliquer sur Ajouter un challenge
  2. Renseigner les informations (image, thème, description, dates)
  3. Cliquer sur Envoyer
